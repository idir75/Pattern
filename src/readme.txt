We should take a stop to think about all the design patterns.

For creational patterns, it focuses on how to create new objects.
We should pay attention to how and when the object is instantiated and return.

For structural patterns, it focused on the relationship among different classes.
It focuses on the concepts like inheritance. 
How to create the system in a most effective structure? How to deal with the relationship in it.
Extends and implements is the core idea here.
Interface, abstraction, implementation, try to find the common things and improve code reuse and maintenance.
